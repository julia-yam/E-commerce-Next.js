### Как запустить:
1. `yarn install` — установка зависимостей.
2. `yarn dev` — запуск локально.

## Переменные окружения
Создайте файл `.env.local`
NEXT_PUBLIC_STRAPI_BASE_URL=[https://front-school-strapi.ktsdev.ru](https://front-school-strapi.ktsdev.ru)
NEXT_PUBLIC_STRAPI_API_URL=[https://front-school-strapi.ktsdev.ru/api](https://front-school-strapi.ktsdev.ru/api)

## Скрипты
| Команда | Действие |
| :--- | :--- |
| `yarn dev` | Запуск сервера разработки |
| `yarn build` | Сборка проекта для продакшена |
| `yarn start` | Запуск собранного приложения |
